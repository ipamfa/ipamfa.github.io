Target Sebelum Lebaran

Jul 23rd, 2014

Beberapa target sebelum lebaran :

  1. dokumentasi konsep dasar lattice, good bases/bad bases
  2. dokumentasi GGH crypto system meliputi proses enkripsi dan dekripsi
  3. dokumentasi rangkuman esensi metode Hastad broadcast attack
  4. memetakan konsep lattice intersection dari pak Willy dari sudut pandang metode Hastad

inti dari broadcast attack hastad adalah mencari solusi untuk sistem kongruen, mirip seperti pada CRT. bedanya pada CRT persamaan kongruen hanya dalam bentuk satu variabel sedangkan pada broadcast attack persamaan berbentuk polynomial

Analisis terhadap konsep CRT (Chinese Remainder Theorem)

  • Problem CRT dapat dilihat sebagai suatu Ring Automorphism
  • Ring sebagai struktur aljabar dengan sifat-sifat tertentu (distributif, komutatif, asosiatif)
  • Ring dapat diekspresikan dalam bentuk additive group dan monoid
  • Jika ring adalah group maka: apa-apa yg berlaku pd group berlaku pula pd ring
  • Pada group berlaku homomorphism (dan juga isomorphism)
  • Introduction to relasi homomorphism dan isomorphism dapat di baca di [Redfield p.160 introduction to abstract algebra]
